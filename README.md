This is a chat build over ExpressJS and WebSockets

## Requirements
- Node JS >= 12
- Any browser
- Grunt
  - Only if you want to modify the styles

## Instalation
1. Clone the repository
2. Create the `.env` file if you want to run the server on other port insted of the `3000``
``` sh
NODE_ENV=production
GUI_PORT=3000
```
3. install via NPM
``` sh
$ npm install
```

## Run
``` sh
$ node server.js
```
